# Python para Análise de Dados — Jornada de Aprendizado

Este repositório documenta minha jornada de aprendizado em Python para análise de dados, inspirado pelos conteúdos e metodologia de ensino de Téo Calvo.

O objetivo é registrar minha evolução prática, consolidar conceitos e desenvolver habilidades aplicadas ao trabalho com dados.

---

## Objetivo do Repositório

- Aprender Python com foco em análise de dados  
- Registrar exercícios, práticas e experimentos  
- Documentar conceitos importantes do aprendizado  
- Desenvolver raciocínio analítico com código  
- Construir base técnica para projetos de dados  

Este repositório representa o início da minha formação prática na área de dados utilizando Python.

---

## O que será desenvolvido aqui

Ao longo do tempo, este espaço reunirá:

- fundamentos de Python  
- manipulação de dados  
- exercícios práticos  
- notebooks exploratórios  
- pequenas análises com dados reais  

A proposta é transformar estudo em prática contínua.

---

## Estrutura inicial do repositório

- `fundamentos-python/` → primeiros conceitos e exercícios  
- `estruturas-de-dados/` → listas, dicionários, loops e lógica  
- `manipulacao-dados/` → primeiros contatos com dados tabulares  
- `notebooks/` → experimentos e práticas guiadas  
- `anotacoes/` → aprendizados e resumos  

---

## Abordagem de Aprendizado

O estudo será orientado por:

1. compreensão da lógica de programação  
2. aplicação prática dos conceitos  
3. resolução de pequenos problemas  
4. contato inicial com dados reais  
5. construção progressiva de análises
